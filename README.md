# OS X Mavericks (10.9)
Theme reproducing the OS X gradients and colors found in the Mavericks release

### Manual installation

Extract the zip file to the themes directory i.e. `/home/USERNAME/.themes`

### Requirements

- GTK+ 3.20 or above
- Murrine and Pixmap theme engines
